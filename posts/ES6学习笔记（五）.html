# 数组新增

1、 for...of循环

   let arr = [1,2,3,4]
   for(let val of arr){
    cosnole.log(val)
   }

   for(let [key,val] of arr.entries()){
    console.log(key,val)
   }

** arr.keys() 数组下标 ** ** arr.entries() 数组某一项 **

2、Array.form 类数组对象转换成数组

  function show(){
    let args = 2、Array.from(arguments)
    args.push（6）
    cosnole.log(args)
  }
  show(1,2,3,4,5)

3、Array.of 一组值转成数组

  let arr = Array.of('apple','banana','peach')
  console.log(arr)

4、find 找到数组中第一个符合条件的数组成员，没找到则返回undefined

  let arr  = [4,5,6,8]
  arr.find((val,index,arr)=>{
    return val>6
  })

5、findIndex 用法同find,找到第一个符合条件的数组下标，没找到则返回-1

6、fill 填充数组

  let arr  = new Array(10)
  arr.fill('1',1,3)
  console.log(arr) //[empty, "1", "1", empty × 7]

7、includes 是否包含某元素，返回布尔值(ES2016,IE不支持)

  let arr = ['apple','banana','peach']
  console.log(arr.includes('apple'))

# 对象新增

1、对象简介语法

  let name = 'apple'
  let price =20
  let  obj = {
    name,
    obj
  }

2、Object.is 比较两个值是否相等

  console.log(Object.is(NaN,NaN)) // true
  console.log(Object.is(+0,-0)) // false
  let obj = {
    name:'apple'
  }
  console.log(Object.is(obj,{...obj})) //false

3、Object.assign() 合并对象

  let obj1= {a:1}
  let obj2={a:2,b:2}
  let obj = Object.assign({},obj1,obj2)
  console.log(obj) //{a: 2, b: 2}

** Object.keys() 遍历对象属性 ** ** Object.values() 遍历对象属性值 ** ** Object.entries() 遍历对象 **
