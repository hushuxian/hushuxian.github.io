# Promise，解决异步回调问题

1、 基本用法

   let a = 1
   let promise = new Promise(function(resolve,reject){
    //立即执行方法
    if(a>10){
      resolve('成功')
    }else{
      reject('失败')
    }
   })
   //promise.then(success,fail)

   promise.then(res=>{
    console.log(res)
   },err=>{ 
    console.log(err)
   })

2、Promise.resolve() 将享幼对象转为fulfilled状态的Promise对象

  Promise.resolve('aaa')
  //等价于
  new Promise(resolve=>resolve('aaa'))

Promise.resolve()参数情况： ①参数是Promise实例，Promise.resolve将不做修改，原封不动返回这个实例 ②参数是一个thenable对象（对象拥有then方法），Promise.resolve会将对象转换为Promise对象，然后立即执行thenable对象的then()方法

  let thenable = {
    then(resolve,reject){
      resolve('aa')
    }
  }
  let p = Promise.resolve(thenable)
  p.then(function(res){
    console.log(res)  //aa
  })

③参数是不具有then()方法的对象，或原始值，则Promise.resolve方法返回一个新的Promise对象，状态为resolved,并将参数传递给新Promise的回调函数

  const p = Promise.resolve({name:'aaa'});

  p.then(function (res) {
    console.log(res) //{name: "aaa"}
  })

④不带有任何参数，直接返回一个resolved状态的 Promise 对象 4、Promise.reject() 返回一个rejected状态的Promise实例 方法的参数，原封不动地作为reject的理由，变成后续方法的参数

  Promise.reject('出错了')
  .catch(e => {
    console.log(e) //出错了
  })

5、Promise.all() 将多个Promise实例包装成一个新的Promise实例， 新的promise实例p的状态： ①只有所有的promise状态都是fulfilled,p的状态才是fulfill，所有被包装的promise返回值组成数组，传递给p的回调函数 ②p1,p2,p3只要有一个rejected,p的状态就是rejected，第一个被reject的实例的返回值，传递给p的回调函数

  let p1 = Promise.resolve('aaa')
  let p2 = Promise.resolve('bbb')
  let p3 = Promise.resolve('ccc')
  let p = Promise.all([p1,p2,p3])
  p.then(res=>{
    console.log(res) //["aaa", "bbb", "ccc"]
  })

  let p1 = Promise.resolve('aaa')
  let p2 = Promise.reject('bbb')
  let p3 = Promise.resolve('ccc')
  let p = Promise.all([p1,p2,p3])
  p.then(res=>{}).catch(err=>{
    console.log(err) //bbb
  })

6、Promise.race() 将多个Promise实例，包装成新的Promise实例，只要其中一个实例改变状态，新的Promise实例状态就跟第一个改变状态的实例一样。参数如果不是Primise实例，则使用Promise.resolve()将参数转为Promise实例，再进一步处理

  let p1 = new Promise(function(resolve,reject){
    setTimeout(()=>{reject('aaa')},1000)
  })
  let p2 = new Promise(function(resolve,reject){
     setTimeout(()=>{reject('bbb')},100)
  })
  let p3 = 'ccc'
  let p = Promise.race([p1,p2,p3])
  p.then(res=>{
    console.log(res) //ccc
  }).catch(err=>{
    console.log(err) 
  })
