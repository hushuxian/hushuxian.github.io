# 声明变量

# var 声明特点：

1、只有函数作用域和全局作用域，无块级作用域

  for(var i = 0;i<10;i++){
    //TODO
  }
  console.log(i) //10

  if(true){
    var a = 0;
  }
  console.log(a) //0

2、变量提升

  var a = 1;
  fn(){
    console.log(a); //undefined
    var a=5;
  }

# ES6 定义：let、const

let定义变量特点： 1、块级作用域内定义，只能在同一块级作用域使用 if(){} for(){} while(){}

  for(let i  = 0;i<10;i++){
    let i = 'aa'
    console.log(i) //'aa' ()父级作用域,{}子级作用域
  }

2、不存在变量提升，在代码块内使用let定义变量，在定义前使用报错

3、同一作用域重复定义报错

4、使用闭包

  let arr= []
  for(let i = 0;i<3;i++){
    arr[i] = function(){
      console.log(i)
    }
  }
  arr[0]() // 0  使用var定义，输出都为3

const定义变量特点：

1、const特性与let一致，除了声明时必须赋值，因为const定义常量，不可重新赋值 2、const定义数组，对象等引用类型,其属性值可改

  const c = [1,2];
  c[0]=22;
  console.log(c); //[22,2]

  const b = {name:'a'};
  b.name='aaaa';
  console.log(b.name)//'aaaa'

定义不可改对象

  let arr = ['a','b'];
  Object.freeze(arr);
  arr.push('c') // 报错
