##计算属性：当模板字符串涉及到的逻辑计算比较复杂时使用。

<div id="app">
        <div ><input type="text" v-model="price1"> <input type="text" v-model="num1"></div>
        <div ><input type="text" v-model="price2"> <input type="text" v-model="num2"></div>
        <div ><input type="text" v-model="price3"> <input type="text" v-model="num3"></div>
        <div>总价:{{price1*num1+price2*num2+price3*num3}}</div>
</div>

let app = new Vue({
    el:"#app",
    data:{
        price1:0,
        price2:0,
        price3:0,
        num1:0,
        num2:0,
        num3:0,
    },
})

使用computed代替

<div>总价:{{total}}</div>

let app = new Vue({
    //...
    computed:{
        total:function(){
            let{price1,num1,price2,num2,price3,num3} = this; //解构赋值获取data里的变量
            return price1*num1+price2*num2+price3*num3;
        }
    }
    
})

computed和methods里定义方法的区别 computed （1）计算属性包含getter和setter js // ... computed: { newprice: { // getter get: function () { return this.price1 + 3 }, // setter set: function (newValue) { this.price1 += 3 } } } // ... （2）基于依赖缓存 （3）依赖属性改变=>触发setter和getter=>计算属性改变 （4）可以依赖其他的计算属性 （5）可依赖其他的数据实例 methods是为实例定义普通的数组，组件重新渲染就会调用 通常一个数据收多个数据的影响时，遍历大数组和做大量计算时使用计算属性。

##watch

    //...
    watch:{

        }

使用handler监听

//...
watch:{
    obj:{
        handler:function(val,oldVal){
            console.log(oldVal.age+'=>'+val.age)
        },
        deep:true, //当监听对象为对象时，deep设为false不能监听对象改变
        imediate:false 
    }
}
