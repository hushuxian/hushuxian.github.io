npm init -y——初始化
npm instll vue 安装vue依赖

2、创建一个vue 实例并挂载到dom上

<div id="app"></div>

let app = new Vue({
    el:"#app",
    data:{ //定义vue实例的数据
            title:"this is demo"
            htmlStr:"<div>html</div>"
    },
    methods:{//定义方法

    }
})

3、模板语法 （1）Mustache语法（双大括号）:可以使用js表达式，不能定义变量，或流程语句

<div>title:{{title}}</div>
<div>{{str.split('')}}</div> √
<div>{{var a = 0}}</div> ×

（2）v-once执行一次绑定，数据再次变化，视图不会发生改变

<div v-once={{title}}></div>

（3）v-bind 标签上属性绑定

<div v-bind={{title}}></div>
``` 可简写成```html
<div :bind={{title}}></div>

（4）插入html语法

<div v-html="htmlStr"></div>

（5）常规内置指令 v-cloak解决页面闪动的问题

<div v-cloak>
    {{title}}
</div>

v-if、v-else-if、v-else、v-show v-on 监听DOM事件

4、vue绑定事件：v-on: 缩写@

<div id="app">   
        <div>{{title}}</div>
        <button v-on:click="changeTitle">change</button>
</div>

let app = new Vue({
    el:"#app",
    data:{
        title:"this is demo"
    },
    methods:{
        changeTitle(e){
            this.title="this is demo2"
        }
    }
    
})

事件修饰符： v-on:click.stop、v-on:submit.prevent、v-on:submit.stop.prevent、v-on:click.capture、v-on:click.self、v-on:click.once、v-on:scroll.passive

按键修饰符：v-on:keyup.enter、v-on:keyup.pagedown

系统修饰符（ctrl\alt\shift\meta）：@keyup.alt.67(alt+ctrl)、@click.ctrl(点击+按下ctrl键)

.exact修饰符：精确控制系统修饰符的组合事件 @click.ctrl.exact(有且只有ctrl被按下的时候触发)

鼠标修饰符：.left、.right、.middle
