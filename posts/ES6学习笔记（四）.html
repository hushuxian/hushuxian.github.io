##箭头函数 1、基本用法

   let show = ()=>1

2、箭头函数特点

  • 改变this指向，指向定义函数所在对象；非箭头函数指向函数运行时对象
  • 箭头函数里没有arguments
  • 箭头函数不能当构造函数

###字符串新增函数 1、includes-字符串查找(ES2016,IE不支持)

  let str = 'I am a student'
  console.log(str.includes('am')) //true

2、startWith/endWith- 是否以xx开头/结尾

  let str = 'I am a student'
  console.log(str.startWith('I')) //true

3、repeat-复制字符串

  let str = 'a'
  console.log(str.repeat(3)) //'aaa'

##函数默认参数 1、默认参数

  function show(int = 0){
    console.log(int)
  }
  show(4) // 4
  show() //0

  function show({x=0,y=0}={}){
    console.log(x,y)
  }
  show({})//0,0
  show()//0,0

##剩余参数 1、接收多个参数

  let show = (...a)=>{
    console.log(a) //[1,2,3]
  }
  show(1,2,3)

1、接收剩余参数

  let show = (a,b,...c)=>{
    console.log(...c) //[3,4,5]
  }
  show(1,2,3,4,5)

...可以用来展开数组

consol.log(...[1,2,3])
