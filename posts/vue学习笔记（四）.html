##不具名插槽

var child = {
    template:`<div><slot>插槽内容</slot></div>`,
  }
  new Vue({
    el:"#app",
    components:{
      'child-component':child
    }
    
})

  <child-component>
    <p>父组件有内容时不显示插槽</p>
  </child-component>

##具名插槽

var child = {
    template:`<div>
    <slot name="top">头部插槽内容</slot>
    <slot>分割线插槽内容</slot>
    <slot name="bottom">底部插槽内容</slot>
    </div>`,
}

 <child-component>
    <template slot="top">占用top插槽的位置，可以定义自己的内容,没有内容时显示插槽内容</template>
    <template>——————分割线——————</template>
 </child-component>

##作用域插槽

var scope = { //子组件的数据绑定在插槽上，在父组件接收子组件传递的数据
    template:`<div>
      <slot v-for="value in list" :value=value></slot>
    </div>`,
    data:function(){
      return{
        list:[1,2,3,4]
      }
    }
}

父组件使用template放置插槽，通过scope="props"接收子组件的数据

<scope-component >
    <template scope="props">
      {{props.value}}
    </template>
</scope-component>
